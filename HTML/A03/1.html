0. 基本规范 所有代码请使用英文的小写 无论路径、文件名。必须使用英文且不能有空格和特殊符号 打开文件，请使用专用的编辑器，编码格式必须为UTF-8 Sublime、VScode、vim（不要用记事本、word文档） 注释的快捷键 ctrl + / （前提是编辑器能识别出你的文件是语言） 1. 注释 不会程序识别并且执行，只是写给程序员看的 不会显示在网页 注释不要去嵌套注释 --> 2. 标签 1. 标签呈现为两种形式 单标签 <标签名> 或 <标签名 /> 双标签 <开始标签> 2. 标签都是闭合的 3. HTML 元素 HTML元素就是标签之间的内容 4. HTML 语法 不区分大小写 但是要求默认小写 5. HTML 灵活性 语法很松散 不会由什么错误提示，只能通过显示效果来判断是否写错 6. HTML 属性 1. 以键值对的方式出现在开始标签内 2. 如果一个标签有多个属性，则使用空格分隔，同时顺序无所谓 3. 属性值可以使用双引号、单引号或不使用，但是默认要求大家使用双引号
